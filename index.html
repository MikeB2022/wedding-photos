One-time setup (~5 min). Uses Google Drive API which works immediately — no verification required.
Already set this up before? Just click Authorize with Google below — your credentials are saved and it reconnects in seconds. Authorization is stored per-browser for security.

Create a Google Cloud project

Go to console.cloud.google.com → New Project → name it Wedding Photos. Then go to APIs & Services → Library, search for Google Drive API and enable it.

Configure OAuth consent screen

Go to APIs & Services → OAuth consent screen → External → fill in app name "Wedding Photos" + your email. Under Scopes add drive.file. Under Test users add your own Google email.

Create OAuth 2.0 credentials

Go to Credentials → Create Credentials → OAuth 2.0 Client ID → Web Application. Add this exact redirect URI:

https://your-site.netlify.app/

Enter credentials

Add Netlify environment variables

Netlify → Site Settings → Environment Variables → add these two, then redeploy:

GOOGLE_CLIENT_ID = (your Client ID)
GOOGLE_CLIENT_SECRET = (your Client Secret)
Google Drive connected! Guest photos will upload to your selected folder.
