Google Photos
Not connected
One-time setup takes about 5 minutes. After this, guest uploads are fully automatic.

Create a Google Cloud project

Go to console.cloud.google.com → New Project → name it Wedding Photos. Then go to APIs & Services → Library, search for Google Photos Library API and enable it.

Configure OAuth consent screen

Go to APIs & Services → OAuth consent screen. Choose External, fill in your app name (e.g. "Wedding Photos"), your email, and save. Under Scopes add .../auth/photoslibrary. Under Test users add your own Google email.

Create OAuth 2.0 credentials

Go to Credentials → Create Credentials → OAuth 2.0 Client ID. Choose Web Application. Under Authorized redirect URIs add exactly:

https://your-site.netlify.app/

Copy the Client ID and Client Secret and paste below.

Enter your credentials

Add environment variables in Netlify

In Netlify go to Site Settings → Environment Variables and add these two variables, then redeploy:

GOOGLE_CLIENT_ID = (paste your Client ID)
GOOGLE_CLIENT_SECRET = (paste your Client Secret)
Google Drive connected! Guest uploads will go directly to your selected Drive folder.
